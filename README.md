# depictio-docs

Documentation for [Depictio](https://github.com/depictio/depictio) - a modern platform for creating interactive dashboards from bioinformatics workflow outputs.

**Live docs:** [depictio.github.io/depictio-docs](https://depictio.github.io/depictio-docs/)

**Key Features Documented:**

- Interactive Dashboards with customizable components
- Cross-Collection Filtering for linked data exploration
- YAML Dashboard Sync for version control and IaC workflows
