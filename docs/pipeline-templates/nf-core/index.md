# nf-core Templates

Templates for [nf-core](https://nf-co.re) pipelines. Each template configures a complete Depictio project — data collections, recipes, dashboards, and cross-DC links — from a single `depictio run` command.

| Template | Pipeline | Versions |
|----------|----------|---------- |
| [ampliseq](ampliseq.md) | 16S / ITS / CO1 amplicon sequencing | 2.14.0, 2.16.0 |
| [viralrecon](viralrecon.md) | Viral assembly and variant calling | 3.0.0 |
